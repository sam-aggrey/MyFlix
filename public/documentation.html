Documentation Page

Building a server-side component of a “movies” web application which will provide users with access to information about different movies, directors, and genres, and also be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request body data format Response body data format
Get a list of movies /movies/

URL http://localhost:8080/movies/		 GET NONE A JSON object holding data about all the movies.
Get data of a single movie by title /movies/[title]

URL http://localhost:8080/movies/Spirited%20Away		 GET NONE A JSON object holding data about a single movie, containing title, director, genre and year property.
Add a new movie /movies/

URL http://localhost:8080/movies/		 POST A JSON object holding data about the movie to add: A JSON object holding data about the movie that was added
Delete a movie /movies/[title]

URL http://localhost:8080/movies/Spirited%20Away		 DELETE NONE A text message indicating whether the movie was successfully removed
